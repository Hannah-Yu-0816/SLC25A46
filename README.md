# SLC26A46

**Author:** Han Yu 
**Last updated:** January 2026  

---

##  Overview
This repository contains a collection of scripts used for **Lack of genetic evidence for a role of SLC25A46 in alpha-synucleinopathies**, including rare variant testing and eQTL-based SMR analysis.  
Each script is designed for reproducibility and easy adaptation to new datasets.

---

##  Repository Structure
| Folder / File | Description |
|----------------|-------------|
| `SKATO.R` | Performs rare variant association testing using SKAT-O |
| `smr.sh` | Runs SMR (Summary-based Mendelian Randomization) analysis |
| `README.md` | General documentation (this file) |
| `METASKAT.R` | Performs meta-analysis of rare variant association results across cohorts using MetaSKAT |


Each script includes detailed comments describing usage, inputs, and parameters.
